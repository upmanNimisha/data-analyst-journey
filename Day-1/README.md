# Data Analysis : examining data to find useful insights and support decision making

Example: A company has data
                  1- Number of website visitors
                  2- Number of people who purchased

# A Data Analyst will:
          Compare both numbers
          Find the issue

--> Insight:
         Many users visit, but few purchase --> possible problem in pricing or website experience


# What does a Data Analyst do?

           --> A Data Analyst typically:
                                     1- Collects data
                                     2- Cleans data (removes errors, missing values)
                                     3- Analyzes data
                                     4- Creates reports/dashboards
                                     5- Gives insights to the business
--> In one line:
    A Data Analyst turns raw data into meaningful insights.

# Types of Data :
                1- Structured Data : 
                                    Organised Data
                                    Easy to Analyze
                                    Eg : SQL Database , Excel sheets 
                2- Unstructured Data :
                                     Unorganised Data 
                                     Hard to Analyze
                                     Eg : Images , Videos < Email
